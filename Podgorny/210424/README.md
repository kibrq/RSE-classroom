# ЗАДАЧА 1
Определите количество различных натуральных значений `n` таких, что значение `F(n, 2)` находится в диапазоне `[100; 1000]`.

```
F(n, m) = 1, m == 0
F(n, m) = n * F(n, m - 1)
```

Ответ: `22`
